# Agents

Brief index of the 3 core agents. See CLAUDE.md for complete documentation.

## Available Agents

### analyzer.md
**Phase 1 Step 2: Codebase Analysis** - Analyzes existing codebase patterns based on approved requirements. Outputs `current_state.md`.

### planner.md
**Phase 2: Planning** - Creates strategic implementation plan with sub-agent research. Outputs `plan.md`, `best-practices.md`, `tasks.md`.

### reviewer.md
**Phase 4: Validation** - Reviews implementation against plan and standards. Outputs `review-report.md`.
