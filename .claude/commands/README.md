# Commands

Brief list of the 5 workflow commands. See CLAUDE.md for complete documentation.

## Available Commands

### /new-session {feature-name}
Initialize new feature session with folder structure and templates

### /research {goal}
**Phase 1** - Two-step process: gather requirements (creates requirement.md), then analyze codebase (creates current_state.md)

### /plan
**Phase 2** - Launch planner agent to create implementation plan (creates plan.md, best-practices.md, tasks.md)

### /implement [phase-X]
**Phase 3** - Execute specific phase of approved plan with real-time task tracking

### /validate
**Phase 4** - Launch reviewer agent to validate implementation (creates review-report.md)
